## Chain-of-Thought Prompting

**Category:** prompting  
**Description:** Ask Copilot to reason step by step before producing the final answer. Improves quality on complex tasks.  
**Example:**
```
// Step 1: Parse the input string into tokens
// Step 2: Validate each token against the schema
// Step 3: Return a structured object or throw an error
```
**Notes:** Explicitly listing steps in comments before the function body tends to produce more accurate and structured code.
