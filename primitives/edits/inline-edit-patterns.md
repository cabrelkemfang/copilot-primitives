## Inline Edit Patterns

**Category:** edits  
**Description:** Common patterns for efficient use of Copilot inline edits (Ctrl+I).  

### Pattern 1 — Explain then Transform
Select code, then prompt:
```
Convert this to async/await
```

### Pattern 2 — Add Error Handling
Select a function, then prompt:
```
Add proper error handling and logging
```

### Pattern 3 — Simplify
Select verbose code, then prompt:
```
Simplify this while keeping the same behavior
```
