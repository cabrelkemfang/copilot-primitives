# Contributing

Contributions are welcome! If you found an interesting Copilot primitive — a pattern, trick, video, or link — feel free to open a PR.

## How to Contribute

1. Fork this repository
2. Add your primitive using the template below
3. Place it in the right folder
4. Open a Pull Request with a clear description

## Primitive Template

```markdown
## Name
**Category:** prompting | agent | edit | chat  
**Description:** What this primitive does and when to use it  
**Example:**
<example code or prompt here>
**Source:** [link] (if from community, otherwise omit)
```

## Folder Guide

| Folder | What goes there |
|--------|----------------|
| `primitives/prompting/` | Prompt patterns and techniques |
| `primitives/agents/` | Agentic / multi-step workflows |
| `primitives/edits/` | Inline edit patterns |
| `primitives/chat/` | Chat context and tricks |
| `community/links.md` | External blog posts or articles |
| `community/videos.md` | Demo or tutorial videos |
| `community/people-to-follow.md` | Creators sharing Copilot content |
